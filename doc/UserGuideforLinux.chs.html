1 简介

HWS 系列采集卡可应用于 4路HDMI，4路SDI 采集卡。采集卡驱动支持Thunderbot3/4/USB4

2 产品安装

  • 关闭计算机电源;

  • 将Hws采集卡插入到计算机PCI Express插槽中/雷电拓展邬中

  • 开启计算机电源，启动操作系统;

3 驱动安装

3.1 驱动安装简介

本节主要介绍如何在 linux 系统环境下安装 Hws Capture 系列采集卡的驱动程序.

3.2 驱动安装

  • 下载驱动程序安装包 hws_XXXX.zip其中 XXXX 表示下载驱动的版本号。

  • 输入命令：unzip hws_XXXX.zip 将驱动解压到当前目录。

  • 解压后包括如下目录和文件：

    • play：play目录中包含了测试采集卡的脚本

    • scripts： scripts 目录中包含了安装、卸载、修复驱动的脚本及系统配置文件。

    • src：src 目录中包含了驱动程序的源码。

    • install.sh：执行该文件即可调用 scripts 目录的中的脚本自动安装驱动程序。

  • 安装内核开发包：安装驱动之前，必须先安装内核开发包，否则安装驱动将失败。

    • Ubuntu：sudo apt-get install linux-headers-`uname -r`

    • Fedora、CentOS、RHEL: sudo yum install kernel-devel

  • 执行当前目录下 install.sh 文件，安装驱动。
